This page uses Turf.js, CartoDB's API layers, some Postgres querying through CartoDB's API, and Leaflet's draw functions. Click on the pin tool and drop a pin anywhere on the map to show data for the closet 5 points in the sidebar. Or use the box tool to find data for all of the points within a custom area

The points visualized are Flickr data that was made available by Eric Fisher. Eric provided datasets for over 50 cities of every Flickr Photo taken over a period of five years. During my time as a research assistant for a Historic Preservation non-profit, I created this visusalization to show which photos in NYC were taken within a Historic Districts. (See Map Legend)

When you drop a pin or draw a box, the map will grab all of the data available from Eric's dataset about the photos nearby. The link for each record will take you over to the user's Flickr account for that photo. I also added the ability to filter the data by year by using the dropdown box in the sidebar.
